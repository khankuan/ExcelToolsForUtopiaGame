ExcelToolsForUtopiaGame
=======================

MS Excel tools created for the game, Utopia, at http://utopia-game.com/

=======================
Excel Simulator
=======================

Excel Simulator is an excel tool to simulate the game, turn by turn (hour by hour). The simulator can be functional without enabling macro, which was used for a reset function.



=======================
Excel Stratsheet
=======================

Excel Stratsheet is an excel tool to calculate statistics given a set of inputs. A solver is also included to generate and optimise inputs.
